# Настройка Яндекс.Почта 360 для домена laser178.ru и SMTP в WordPress

## Цель

Создать почтовый ящик `info@laser178.ru`, получить пароль приложения и подключить его к плагину **WP Mail SMTP** на сайте `laser178.ru`.

---

## Что понадобится

- Доступ к панели управления DNS домена (Beget).
- Яндекс ID (аккаунт Яндекса).
- Доступ к админ-панели WordPress (`laser178.ru/wp-admin`).

---

## Этап 1. Подключение домена к Яндекс.Почта 360

### Шаг 1.1. Перейдите в сервис

1. Откройте [https://360.yandex.ru/pochtadoma/](https://360.yandex.ru/pochtadoma/).
2. Нажмите **«Подключить»** или **«Войти»**.
3. Войдите под своим Яндекс ID.

### Шаг 1.2. Добавьте домен

1. В личном кабинете выберите **«Добавить домен»**.
2. Введите: `laser178.ru`.
3. Выберите способ подтверждения: **DNS-запись** (рекомендуется) или **HTML-файл**.

### Шаг 1.3. Подтвердите владение доменом через DNS

1. Яндекс покажет уникальную TXT-запись для подтверждения. Обычно это что-то вроде:
   ```
   yandex-verification: xxxxxxxxxxxxxxxx
   ```
2. Скопируйте значение.
3. Зайдите в панель управления **Beget** → **Управление доменами** → **DNS-редактор** (или **DNS-записи**).
4. Добавьте TXT-запись:
   - **Имя/Хост:** `@` или `laser178.ru`
   - **Значение:** скопированная строка из Яндекса
   - **TTL:** по умолчанию (3600)
5. Сохраните.
6. Вернитесь в кабинет Яндекс.Почта 360 и нажмите **«Проверить»**.
7. Подождите 5–30 минут, пока DNS обновится.

> **Важно:** значение TXT-записи у каждого пользователя своё. Берите его из кабинета Яндекс.Почта 360, а не из этой инструкции.

---

## Этап 2. Настройка MX-записей для приёма писем

### Шаг 2.1. Добавьте MX-запись в Beget

1. В панели Beget перейдите в **DNS-редактор** для домена `laser178.ru`.
2. Добавьте MX-запись:
   - **Имя/Хост:** `@` или `laser178.ru`
   - **Значение/Почтовый сервер:** `mx.yandex.net`
   - **Приоритет:** `10`
   - **TTL:** по умолчанию
3. Если есть старые MX-записи от Beget или других сервисов — удалите их, чтобы осталась только запись Яндекса.
4. Сохраните.

### Шаг 2.2. Добавьте SPF-запись (рекомендуется)

1. Добавьте TXT-запись:
   - **Имя/Хост:** `@`
   - **Значение:** `v=spf1 redirect=_spf.yandex.net`
   - **TTL:** по умолчанию
2. Сохраните.

### Шаг 2.3. Добавьте DKIM (по желанию, для лучшей доставки)

1. В кабинете Яндекс.Почта 360 включите **DKIM-подпись**.
2. Яндекс сгенерирует открытый ключ и имя TXT-записи (обычно `mail._domainkey`).
3. Скопируйте значение и добавьте TXT-запись в Beget.
4. Вернитесь в кабинет Яндекс и нажмите **«Проверить»**.

> Без DKIM письма тоже будут отправляться, но с DKIM вероятность попадания в спам ниже.

---

## Этап 3. Создание почтового ящика

1. В кабинете Яндекс.Почта 360 откройте раздел **«Пользователи»** или **«Почтовые ящики»**.
2. Нажмите **«Добавить пользователя»**.
3. Укажите:
   - **Имя:** `info`
   - **Почта:** `info@laser178.ru`
   - **Пароль:** придумайте сложный пароль (он понадобится только для входа в веб-почту)
4. Сохраните.
5. Запомните или запишите пароль.

---

## Этап 4. Создание пароля приложения

Пароль приложения — это отдельный токен, который WordPress будет использовать для отправки писем. Он отличается от основного пароля ящика.

1. Войдите в Яндекс.Почту под `info@laser178.ru`: [https://mail.yandex.ru/](https://mail.yandex.ru/).
2. В правом верхнем углу нажмите на **имя аккаунта** → **«Управление аккаунтом»** или перейдите на [https://id.yandex.ru/](https://id.yandex.ru/).
3. Выберите раздел **«Безопасность»** → **«Пароли приложений»** (или **«Пароли и авторизация»** → **«Приложение пароли»**).
4. Нажмите **«Создать пароль приложения»**.
5. Введите название: `WordPress SMTP`.
6. Нажмите **«Создать»**.
7. **Скопируйте полученный пароль**. Его нельзя будет посмотреть повторно. Если потеряете — придётся создавать новый.

---

## Этап 5. Настройка WP Mail SMTP в WordPress

1. Войдите в админ-панель WordPress: `https://laser178.ru/wp-admin`.
2. Перейдите в **WP Mail SMTP** → **Настройки**.
3. В разделе **«From Email»** введите: `info@laser178.ru`.
4. В разделе **«From Name»** введите: `Лазер Антикор`.
5. В поле **«Mailer»** выберите **«Other SMTP»**.
6. Заполните поля:
   | Поле | Значение |
   |------|----------|
   | **SMTP Host** | `smtp.yandex.ru` |
   | **Encryption** | **SSL** |
   | **SMTP Port** | `465` |
   | **Auto TLS** | **ON** |
   | **Authentication** | **ON** |
   | **SMTP Username** | `info@laser178.ru` |
   | **SMTP Password** | скопированный пароль приложения |
7. Нажмите **«Save Settings»**.
8. Перейдите на вкладку **«Email Test»**.
9. Введите тестовый email (например, `Evgprohorov77@gmail.com`) и нажмите **«Send Email»**.
10. Проверьте почту. Если письмо пришло — SMTP настроен.

---

## Что делать, если тестовое письмо не доходит

1. Проверьте, что пароль приложения скопирован полностью и без пробелов.
2. Убедитесь, что в поле **From Email** указан именно `info@laser178.ru` (не другой адрес).
3. Проверьте папку **Спам**.
4. Подождите 1–2 часа после добавления MX-записей — иногда DNS обновляется не сразу.
5. Включите логирование в WP Mail SMTP: **Настройки** → **Debug** → **Enable Logging**, повторите отправку и посмотрите ошибку.

## Результат

После выполнения всех шагов WordPress сможет отправлять письма с адреса `info@laser178.ru` через SMTP Яндекса. Это нужно для:
- форм обратной связи,
- уведомлений о записях,
- писем клиентам,
- транзакционных писем.
