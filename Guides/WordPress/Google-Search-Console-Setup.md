# Добавление сайта laser178.ru в Google Search Console

## Цель

Подтвердить права на домен `laser178.ru` и добавить сайт в [Google Search Console](https://search.google.com/search-console) для индексации в Google.

---

## Что понадобится

- Google-аккаунт (Gmail).
- Доступ к панели управления DNS домена (Beget) **или** доступ к FTP/админке сайта.

---

## Этап 1. Вход в Google Search Console

1. Откройте [https://search.google.com/search-console](https://search.google.com/search-console).
2. Войдите под своим Google-аккаунтом.

---

## Этап 2. Добавление сайта

### Вариант A. Подтверждение через домен (рекомендуется)

1. В поле **«Домен»** введите: `laser178.ru`.
2. Нажмите **«Продолжить»**.
3. Google предложит TXT-запись для подтверждения. Скопируйте её.
4. Зайдите в панель **Beget** → **DNS-редактор** для `laser178.ru`.
5. Добавьте TXT-запись:
   - **Имя/Хост:** `@` или `laser178.ru`
   - **Значение:** скопированная строка из Google Search Console
   - **TTL:** по умолчанию (3600)
6. Сохраните.
7. Вернитесь в Google Search Console и нажмите **«Подтвердить»**.
8. Подождите несколько минут, пока DNS обновится.

### Вариант B. Подтверждение через URL-префикс

1. Выберите вкладку **«URL-префикс»**.
2. Введите: `https://laser178.ru/`.
3. Нажмите **«Продолжить»**.
4. Выберите способ **«HTML-файл»**.
5. Скачайте файл или скопируйте его имя, например:
   ```
   googleXXXXXXXXXXXXXXXX.html
   ```
6. Передайте имя файла и содержимое ассистенту — файл будет загружен в корень сайта.
7. Вернитесь в Google Search Console и нажмите **«Подтвердить»**.

---

## Этап 3. Добавление Sitemap

После подтверждения прав:

1. В левом меню выберите **«Sitemap»**.
2. Введите адрес:
   ```
   sitemap_index.xml
   ```
3. Нажмите **«Отправить»**.
4. Подождите несколько дней — Google проиндексирует страницы.

---

## Этап 4. Проверка URL

1. В верхней строке поиска Google Search Console введите:
   ```
   https://laser178.ru/
   ```
2. Нажмите **«Проверить URL»**.
3. Если URL не в индексе — нажмите **«Запросить индексацию»**.
4. Повторите для основных страниц:
   - `https://laser178.ru/about/`
   - `https://laser178.ru/services/`
   - `https://laser178.ru/contacts/`

---

## Рекомендуемые настройки

1. **Географический таргетинг** — в настройках сайта укажите **Россия**.
2. **Проверка Core Web Vitals** — убедитесь, что страницы загружаются быстро.
3. **Мобильная версия** — проверьте, что нет ошибок мобильной usability.
4. **Ссылки** — в разделе **«Ссылки»** можно отслеживать внешние ссылки на сайт.

---

## Что делать дальше

- Установить **Яндекс.Метрику** для отслеживания посещаемости.
- Опубликовать первые кейсы для ускорения индексации.
- Подключить **Google Analytics** (по желанию).
