# Template: SEO-статья «База по моделям»

## Структура

1. H1 — «Антикор {{brand}} {{model}} в {{region}}»
2. Введение: почему эта модель подвержена коррозии
3. Слабые места / зоны риска
4. Рекомендуемый комплекс работ
5. Сколько стоит (только public цены)
6. FAQ (3–5 вопросов)
7. CTA + контакты

## Правила

- Использовать `Knowledge/Company/cars.yaml`.
- Если нет данных по модели — использовать generic описание с `[needs verification]`.
- Регион: `company.city_for_seo`.
- Цены: только `public: true` из `prices.yaml`.
- H2: ключевые слова с названием модели и города.
