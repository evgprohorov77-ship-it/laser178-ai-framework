# Template: Страница прайса

## Структура

1. H1 — «Стоимость услуг антикора и лазерной очистки в {{region}}»
2. Дисклеймер: цены зависят от состояния, модели, объёма работ
3. Таблица цен (только public + verified)
4. Калькулятор / как получить точную стоимость
5. FAQ по ценообразованию
6. CTA + контакты

## Правила

- Использовать только `public: true` цены из `Knowledge/Company/prices.yaml`.
- Не использовать «от ... ₽», если `client_facing_text` запрещает.
- Если цена не public — писать «уточняйте по телефону».
- Указывать единицы (м², услуга, снятие+установка).
