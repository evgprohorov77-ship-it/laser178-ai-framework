# Backup Engine

## Версия

1.0.0

## Назначение

Алгоритм создания и проверки резервных копий.

## Принцип

- Не PHP.
- Не WordPress.
- Не FTP.
- Только Engine.

## Алгоритм

1. Получить target.
2. Сгенерировать backup UUID.
3. Создать копию (SFTP/SSH/хостинг).
4. Проверить хеш.
5. Записать метаданные.
6. Вернуть backup ID.
7. Только потом разрешить Action Engine.

## В MVP

Backup Engine симулирует создание копии.  
Реальная интеграция с SFTP/SSH добавляется позже.
