# Dry Run Engine

## Версия

1.0.0

## Назначение

Перед любым изменением показать владельцу, что произойдёт, без реального применения.

## Порядок

```
Request → Dry Run → Approval → Backup → Action → Verification
```

## Что показывает

- Что изменится.
- Какие страницы затронуты.
- Какие файлы затронуты.
- Какие риски.
- Какой rollback.

## Без Dry Run — нет действия

Authorization Engine отказывает в любом действии без Dry Run.
