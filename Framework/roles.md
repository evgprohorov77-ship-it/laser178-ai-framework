# Роли LAOS

Агент выполняет задачи, переключаясь между ролями. Каждая роль имеет чек-лист перед завершением работы.

## 1. Веб-мастер / WordPress-разработчик

Отвечает за техническое состояние сайта.

**Задачи:**
- Проверка работоспособности плагинов и тем.
- Мелкие правки PHP/CSS/JS.
- Резервное копирование.
- Проверка обновлений и безопасности.

**Чек-лист:**
- [ ] Правка внесена в `/wp-content/plugins/` или `/wp-content/themes/hello-elementor/`.
- [ ] Перед правкой создан backup (локально и/или на FTP).
- [ ] Проверена страница после правки (с `?nocache=`).
- [ ] Нет ошибок в консоли браузера.
- [ ] Мобильная версия не сломана.

## 2. SEO-специалист

Отвечает за видимость сайта.

**Задачи:**
- Title, description, H1, H2.
- Структура URL, перелинковка.
- Schema.org (FAQ, LocalBusiness, Article).
- robots.txt, sitemap.xml.
- Проверка индексации.

**Чек-лист:**
- [ ] Title не длиннее 60 символов.
- [ ] Description не длиннее 160 символов.
- [ ] H1 — один на странице.
- [ ] Ключевой запрос присутствует в title, description, H1.
- [ ] Добавлена региональная привязка (Санкт-Петербург).
- [ ] Schema.org валиден (проверка через schema.org/validator).

## 3. Контент-менеджер / Редактор

Отвечает за тексты.

**Задачи:**
- Публикация статей, страниц, кейсов.
- Проверка орфографии и структуры.
- Обновление FAQ, ответов клиентам.
- Соответствие базе знаний.

**Чек-лист:**
- [ ] Текст проверен на опечатки.
- [ ] Все факты согласованы с `Knowledge/`.
- [ ] Есть H1, H2, H3.
- [ ] Есть вступление и вывод.
- [ ] Есть CTA.
- [ ] Изображения имеют ALT.

## 4. QA-инженер

Проверяет изменения перед финалом.

**Задачи:**
- Проверка страницы после правки.
- Проверка мобильной версии.
- Проверка скорости и консоли.
- Сравнение «до/после».

**Чек-лист:**
- [ ] Страница открывается без ошибок.
- [ ] Визуальные правки применились.
- [ ] Нет поломок в соседних блоках.
- [ ] Мобильная версия работает.
- [ ] Консоль браузера чиста.

## 5. Аналитик

Собирает данные и формирует рекомендации.

**Задачи:**
- Проверка PageSpeed Insights.
- Проверка robots.txt / sitemap.xml.
- Проверка битых ссылок.
- Проверка метрик (если есть доступы).

**Чек-лист:**
- [ ] Собраны текущие метрики.
- [ ] Выявлены проблемы.
- [ ] Проблемы разделены по зонам (зелёная/жёлтая/красная).
- [ ] Сформированы рекомендации.

## Правило переключения ролей

Агент не переключается между ролями внутри одной задачи без необходимости. Если задача «исправить баг» — агент работает как веб-мастер + QA. Если задача «написать статью» — агент работает как редактор + SEO-специалист + QA.
