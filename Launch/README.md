# Launch

## Версия

1.0.0

## Назначение

Каталог `Launch/` — единое место для управления запуском сайта `https://laser178.ru`.  
Содержит чек-листы, отчёты и артефакты релиза.

## Режим работы

- **Feature freeze ядра LAOS сохраняется.**
- Не создаём новые Engine, Adapter, Layer, абстракции.
- Разрешено: исправлять ошибки, писать контент, править SEO, заполнять Knowledge, готовить сайт к запуску.

## Структура

| Файл | Назначение |

| `Release_Report.md` | Итоговый отчёт о релизе (заполняется после запуска) |

## Как работать с чек-листами

- Каждый пункт имеет статус: `TODO`, `IN PROGRESS`, `BLOCKED`, `DONE`.
- Каждый пункт имеет владельца и приоритет.
- Перед началом задачи Hermes проверяет: связана ли она с запуском сайта, повышает ли готовность, можно ли отложить после запуска.
- Идеи, не влияющие на запуск, переносятся в `Roadmap/PostLaunch/`.

## Статусные лейблы

```markdown
- [ ] TODO — ещё не начато
- [>] IN PROGRESS — в работе
- [!] BLOCKED — заблокировано, нужен ввод от владельца или третьей стороны
- [x] DONE — выполнено и проверено
```
