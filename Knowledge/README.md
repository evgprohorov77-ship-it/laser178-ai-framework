# Knowledge

## Структура

База знаний разбита на отдельные файлы по доменам, чтобы каждый документ оставался небольшим и поддерживаемым.

| Файл | Содержание |
|------|------------|
| `Company/company.md` | Общие сведения о компании. |
| `Company/employees.md` | Сотрудники и роли. |
| `Company/contacts.md` | Контакты и адреса. |
| `Company/services.md` | Услуги и пакеты. |
| `Company/materials.md` | Материалы и ценообразование. |
| `Company/equipment.md` | Оборудование. |
| `Company/prices.md` | Принципы ценообразования и прайсы. |
| `Company/faq.md` | FAQ и возражения. |
| `Company/guarantees.md` | Гарантии. |
| `Company/partners.md` | Партнёры и поставщики. |
| `Company/cars.md` | Целевые модели автомобилей. |

## Преимущества разделения

- Меньше конфликтов при редактировании.
- Проще искать.
- Можно обновлять отдельные разделы без затрагивания всего файла.
- LAOS может загружать только нужный контекст.
