# Research: Competitors

## Статус

- **Не является частью Knowledge/Company.**
- **Цель:** временные исследовательские заметки.
- **Каждая запись содержит дату и источник.**

## Формат записи

```markdown
## <Название>

- **Дата:** YYYY-MM-DD
- **Источник:** URL / личный осмотр / разговор
- **Расположение:** город/район
- **Услуги:** ...
- **Цены:** ...
- **Сильные стороны:** ...
- **Слабые стороны:** ...
- **Примечания:** ...
```

## Правила

- Не копировать цены без источника.
- Не делать утверждений о качестве без доказательств.
- Обновлять дату при изменении данных.
- Не использовать конкурентов как источник для собственных цен/условий.

## Примечание

Файл-заглушка. Исследования будут добавляться по запросу владельца или в рамках отдельных задач.
