# Severity Classification

## Уровни критичности

### P0 — Критическая

- Сайт может перестать работать.
- Утечка данных, взлом, недоступность админки.
- **Автоматическое исправление:** запрещено.
- **Действие:** Issue + немедленное уведомление владельца.

### P1 — Высокая

- Безопасность, повреждение SEO, потеря данных.
- **Автоматическое исправление:** запрещено без подтверждения.
- **Действие:** запросить подтверждение владельца.

### P2 — Средняя

- SEO, структура, скорость, UX.
- **Автоматическое исправление:** разрешено при confidence ≥ 0.70 и наличии rollback.
- **Действие:** autofix + verify.

### P3 — Низкая

- Рекомендации, улучшения, орфография, ALT, описание.
- **Автоматическое исправление:** разрешено для очевидных правок.
- **Действие:** autofix + verify или Issue.

## Confidence

- `0.90–1.00` — высокая уверенность, можно исправлять автоматически.
- `0.70–0.89` — достаточная уверенность, autofix разрешён для P2/P3.
- `0.50–0.69` — низкая уверенность, требуется ручная проверка.
- `0.00–0.49` — очень низкая уверенность, отклонить как ложное срабатывание или Issue.

## Правило

Если `confidence < 0.70` — автоматическое исправление запрещено.
