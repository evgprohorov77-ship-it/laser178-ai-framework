# laser178-ai-framework

Фреймворк для взаимодействия GPT → Hermes → сайт laser178.ru.

## Идея

- **GPT** выступает в роли аналитика/контент-менеджера: находит ошибки, предлагает тексты, пометки, SEO-идеи.
- **Hermes** выполняет технические правки на сайте и отчитывается в issue.
- **GitHub Issues** — единый источник задач, история и чек-листы.

## Как использовать

1. GPT создаёт issue по одному из шаблонов.
2. Hermes забирает открытые issues, делает правки, пишет комментарий и закрывает issue.
3. Владелец (Евгений) может пересмотреть/переоткрыть issue в любой момент.

## Шаблоны issue

| Шаблон | Когда использовать |
|--------|------------------|
| `bugfix.md` | Визуальный баг, битая ссылка, ошибка в JS/CSS |
| `content.md` | SEO, тексты, заголовки, description, статьи |
| `feature.md` | Новые блоки, калькуляторы, интеграции |
